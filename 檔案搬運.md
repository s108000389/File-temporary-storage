### ls

```
按照檔案大小排序 : ls -ls
加上 -h 參數可以上檔案的大小資訊以更容易被閱讀的形式表示： ls -lsh
按照檔案修改時間排序 : ls -lt
反向排序 : 加上r 例如 ls -ltr

```
### 目錄的建立與刪除
```
建立一個『空目錄』 $ mkdir
『刪除空目錄』 $  rmdir

複製檔案 $ cp
-p 參數來保留權限和擁有者及群組
  例 : 將 /etc/hosts 複製到本目錄
       cp /etc/hosts .
       將 /etc 整個資料複製到本目錄
       cp /etc .; cp -r /etc .; cp -a /etc .
刪除檔案 $ rm
檔名的移動及更名 $ mv (move)

```
